# dearpygui_ros
Sample proglams of DearPyGui + ROS1

# Required library

  $ pip3 install dearpygui

# Howto test

  $ roscore
  $ rosrun dearpygui_ros dearpygui_talker
  $ rosrun dearpygui_ros dearpygui_listener
